##そもそも酸素分圧とは(PiO2)吸った域の酸素分圧

- 吸った息の酸素分圧は気圧に左右される。

* よって、(大気圧-47)×酸素濃度で計算されます。
* この「47」という数字は、体温37度と仮定したときに、空気を100%加湿したときの水蒸気圧のことです。吸い込んだ空気は渇いたままでなく、しっかりと加湿されてから気道を通ります、その分の圧力を引く必要があり、大気中の酸素濃度は21%として
* 標高0mのところにいれば吸った息の酸素分圧は150mmHgでした。
